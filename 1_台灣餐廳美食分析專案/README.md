# 🍽️ 台灣餐廳美食數據分析專案

## 專案簡介

本專案分析台灣四大城市（台北、台中、台南、高雄）的餐廳數據，包含評分、價格、餐廳類型等指標，透過豐富的視覺化圖表發現美食趨勢。

### ✨ 功能特色

- **評分分析**：評分分布、城市評分比較、餐廳類型評分
- **價格分析**：價格分布、價格與評分關係、城市消費比較
- **類型分析**：餐廳類型分布、各城市類型偏好、評論數比較
- **城市比較**：餐廳數量、平均評分、價格區間、高評價比例
- **相關性分析**：評分、評論數、價格之間的相關性
- **本地數據**：內建模擬數據，無需網路連線
- **完整報表**:輸出 PNG 圖表和 PDF 報告

### 🏙️ 分析城市

- 🔵 台北
- 🟢 台中
- 🟡 台南
- 🔴 高雄

### 🍜 餐廳類型

台式料理、日式料理、義式料理、美式料理、中式料理、韓式料理、泰式料理、咖啡廳、火鍋、燒烤、小吃、甜點店

### 練習 2：推薦系統

根據以下條件推薦餐廳：
- 指定城市
- 價格預算
- 餐廳類型
- 最低評分

```python
def recommend_restaurant(city, max_price, cuisine_type, min_rating):
    result = df[(df['City'] == city) &
                (df['Avg_Price'] <= max_price) &
                (df['Cuisine_Type'] == cuisine_type) &
                (df['Rating'] >= min_rating)]
    return result.sort_values('Rating', ascending=False)
```
