## 專案概述

### 🎯 專案目標

透過分析多個社群媒體平台的數據，學習如何：
- 處理時間序列數據
- 計算成長率和互動率
- 建立專業的視覺化儀表板
- 進行多平台比較分析
- 生成完整的分析報告

### ⏱ 預計時間

- **環境安裝：** 10-15 分鐘
- **數據生成：** 1-2 分鐘
- **執行分析：** 2-3 分鐘
- **學習理解：** 60-90 分鐘

### 📊 分析內容

本專案會分析以下指標：
- 粉絲數成長趨勢
- 總成長率
- 平均互動率
- 貼文頻率
- 互動類型分佈（按讚、分享、留言）
- 平台間比較

## 圖表解讀說明

### 📊 儀表板圖表（第一組）

#### 圖 1：粉絲數成長趨勢

**如何解讀：**
1. **觀察趨勢方向**
   - 向上 = 粉絲數持續成長 ✅
   - 向下 = 粉絲數減少 ❌
   - 平穩 = 成長停滯 ⚠️

2. **比較成長速度**
   - 曲線越陡 = 成長越快
   - 比較不同平台的斜率

3. **找出成長最快的平台**
   - 通常 TikTok 和 YouTube 成長最快
   - LinkedIn 成長較慢

**實際應用：**
- 如果某平台成長停滯，需要調整策略
- 成長快的平台可以投入更多資源

---

#### 圖 2：總成長率比較

**如何解讀：**
1. **長條長度 = 成長幅度**
   - 越長 = 成長越多
   - 比較不同平台的成長表現

2. **百分比數值**
   - 20% = 粉絲數增加 20%
   - 50% = 粉絲數增加 50%（翻倍）

3. **找出最佳表現**
   - 通常 TikTok 成長率最高（約 50%）
   - LinkedIn 成長率最低（約 7%）

**實際應用：**
- 成長率高的平台值得重點經營
- 成長率低的平台需要檢討策略

---

#### 圖 3：平均互動率比較

**如何解讀：**
1. **互動率意義**
   - 互動率 = (互動數 / 粉絲數 / 貼文數) × 100%
   - 反映內容的受歡迎程度

2. **平台差異**
   - TikTok：約 8%（最高）
   - Instagram：約 5%
   - Facebook：約 3%
   - LinkedIn：約 1.5%（最低）

3. **品質指標**
   - 互動率高 = 內容品質好
   - 互動率低 = 需要改善內容

**實際應用：**
- 互動率是衡量內容品質的重要指標
- 不同平台有不同的基準值

---

#### 圖 4：貼文頻率 vs 總互動數

**如何解讀：**
1. **散佈圖模式**
   - 觀察點的分佈位置
   - 找出最佳貼文頻率

2. **圓圈大小**
   - 越大 = 粉絲數越多
   - 通常粉絲多 = 總互動數高

3. **關係分析**
   - 貼文越多 ≠ 互動越多
   - 品質比數量更重要

**實際應用：**
- 找出最有效率的貼文頻率
- 避免過度發布導致互動率下降

## 數據分析技巧

### 技巧 1：計算成長率

**公式：**
```
總成長率 = ((最終值 - 初始值) / 初始值) × 100%
```

**範例：**
- 初始粉絲數：50,000
- 最終粉絲數：60,000
- 成長率 = ((60,000 - 50,000) / 50,000) × 100% = 20%

---

### 技巧 2：計算互動率

**公式：**
```
互動率 = (總互動數 / 粉絲數 / 貼文數) × 100%
```

**範例：**
- 總互動數：2,000
- 粉絲數：50,000
- 貼文數：2
- 互動率 = (2,000 / 50,000 / 2) × 100% = 2%

---

### 技巧 3：移動平均計算

**目的：** 平滑短期波動，顯示長期趨勢

**公式：**
```
7日移動平均 = (第1天 + 第2天 + ... + 第7天) / 7
```

**Python 實作：**
```python
df['MA7'] = df['Total_Engagement'].rolling(window=7).mean()
```

---

### 技巧 4：平台比較

**方法：**
1. 標準化數據（以第一天為基準 = 100）
2. 計算相對成長
3. 比較不同平台的表現

**Python 實作：**
```python
normalized = (df['Followers'] / df['Followers'].iloc[0]) * 100
```
