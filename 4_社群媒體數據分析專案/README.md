# 📱 社群媒體數據分析專案

## 專案簡介

本專案提供完整的社群媒體數據分析工具，分析多個主流社群平台的表現，包含粉絲成長、互動率、貼文頻率等關鍵指標，並產生專業的視覺化儀表板和統計報告。

### ✨ 功能特色

- **多平台分析**：支援 Facebook、Instagram、Twitter、YouTube、TikTok、LinkedIn
- **粉絲成長分析**：追蹤各平台粉絲數成長趨勢
- **互動率分析**：計算按讚、分享、留言等互動指標
- **平台比較**：多平台表現對比分析
- **視覺化儀表板**：2x2 綜合分析圖表 + 詳細分析圖表
- **詳細統計報告**：輸出完整數據摘要和排行榜
- **本地數據**：內建模擬數據，無需網路連線
- **完整報表**：輸出 PNG 圖表和 PDF 報告

### 📊 分析平台

- 📘 **Facebook** - 基準粉絲 5 萬，每日成長 0.05%
- 📷 **Instagram** - 基準粉絲 8 萬，每日成長 0.08%
- 🐦 **Twitter** - 基準粉絲 3 萬，每日成長 0.03%
- 📺 **YouTube** - 基準粉絲 12 萬，每日成長 0.1%
- 🎵 **TikTok** - 基準粉絲 15 萬，每日成長 0.12%
- 💼 **LinkedIn** - 基準粉絲 2 萬，每日成長 0.02%

**數據年份：2026 年全年**

### 指標說明

#### 總成長率
- **計算方式**：((最終粉絲數 - 初始粉絲數) / 初始粉絲數) × 100%
- **意義**：反映平台在一年內的整體成長幅度

#### 平均每日成長
- **計算方式**：每日粉絲數變化的平均值
- **意義**：反映平台每日的成長速度

#### 平均互動率
- **計算方式**：(總互動數 / 總粉絲數 / 總貼文數) × 100%
- **意義**：反映內容的受歡迎程度和粉絲活躍度

#### 總互動數
- **計算方式**：按讚數 + 分享數 + 留言數的總和
- **意義**：反映平台的整體互動表現

### 數據特點

- **時間範圍**：2026年全年（365天）
- **數據類型**：模擬數據，基於真實社群媒體行為模式
- **更新頻率**：每日數據
- **平台數量**：6個主流平台
- **數據完整性**：無缺失值，可直接分析

### 數據生成邏輯

#### 粉絲成長
- 使用指數成長模型：`followers = base × e^(growth_rate × days)`
- 加入隨機波動（±2%）模擬真實情況
- 確保粉絲數單調遞增（不會減少）

#### 互動率
- 不同平台有不同的基準互動率
- 加入隨機波動（-30% 至 +50%）
- 反映內容品質的變化

#### 貼文頻率
- 平日：1-4 篇（隨機）
- 週末：0-2 篇（較少）
- 模擬真實的發布習慣

#### 互動類型分佈
- 按讚：約 70%（最容易）
- 分享：約 15%（中等）
- 留言：約 15%（最難）

3. **Linux：** 安裝中文字型套件

**手動修改字型：**
開啟 `main.py`，找到第 23 行：
```python
plt.rcParams['font.sans-serif'] = ['Microsoft JhengHei', 'Arial Unicode MS', 'sans-serif']
```

改為您系統中有的字型：
```python
plt.rcParams['font.sans-serif'] = ['您的字型名稱', 'sans-serif']
```

platform_configs = [

PLATFORMS = [..., '新平台名稱']

PLATFORM_NAMES = {
