Unit 3: Output Parsers

將 AI 輸出轉換為結構化數據

JSON Parser Pydantic Parser List Parser

為什麼需要 Output Parser？

AI 預設輸出是純文字，但我們需要結構化數據來進行後續處理。

原始 AI 輸出

台積電目前股價約 580 元，
建議評級為「買入」，
目標價位是 650 元。

結構化輸出

{
  "stock": "台積電",
  "price": 580,
  "rating": "買入",
  "target": 650
}
重點：結構化數據可以直接用於程式運算、存入資料庫、或傳給前端顯示。

基礎：StrOutputParser

最簡單的解析器，直接取出 AI 回覆的文字內容：

from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser

llm = ChatOpenAI(model="gpt-3.5-turbo")
parser = StrOutputParser()

# 建立 Chain
chain = llm | parser

# 執行
result = chain.invoke("用一句話介紹 Python")
print(result)  # 直接是字串，不是 AIMessage 物件
print(type(result))  # <class 'str'>

實用：JsonOutputParser

from langchain_core.output_parsers import JsonOutputParser
from langchain_core.prompts import PromptTemplate

parser = JsonOutputParser()

template = PromptTemplate(
    template="""分析以下餐廳評論，輸出 JSON 格式。
{format_instructions}

評論：{review}""",
    input_variables=["review"],
    partial_variables={"format_instructions": parser.get_format_instructions()}
)

chain = template | llm | parser

result = chain.invoke({
    "review": "這家牛肉麵超讚！湯頭濃郁，肉給很多，下次還會再來"
})

print(result)
# {'sentiment': 'positive', 'food_quality': 'excellent', 'would_revisit': True}

進階：PydanticOutputParser

使用 Pydantic 定義資料結構，確保輸出格式正確：

from langchain_core.output_parsers import PydanticOutputParser
from pydantic import BaseModel, Field
from typing import List

# 定義資料結構
class StockAnalysis(BaseModel):
    stock_name: str = Field(description="股票名稱")
    current_price: float = Field(description="目前股價")
    rating: str = Field(description="評級：買入/持有/賣出")
    reasons: List[str] = Field(description="分析理由")

parser = PydanticOutputParser(pydantic_object=StockAnalysis)

# 取得格式說明
print(parser.get_format_instructions())

實作：股票分析結構化輸出

from langchain_core.prompts import PromptTemplate

template = PromptTemplate(
    template="""你是股票分析師。分析 {stock} 並以指定格式輸出。
{format_instructions}""",
    input_variables=["stock"],
    partial_variables={"format_instructions": parser.get_format_instructions()}
)

chain = template | llm | parser

# 執行分析
result = chain.invoke({"stock": "台積電"})

# result 是 StockAnalysis 物件
print(f"股票：{result.stock_name}")
print(f"股價：{result.current_price}")
print(f"評級：{result.rating}")
print(f"理由：")
for r in result.reasons:
    print(f"  - {r}")

列表解析：CommaSeparatedListOutputParser

from langchain_core.output_parsers import CommaSeparatedListOutputParser

parser = CommaSeparatedListOutputParser()

template = PromptTemplate(
    template="列出 5 個 {category} 相關的關鍵字\n{format_instructions}",
    input_variables=["category"],
    partial_variables={"format_instructions": parser.get_format_instructions()}
)

chain = template | llm | parser

result = chain.invoke({"category": "資料視覺化"})
print(result)
# ['Matplotlib', 'Seaborn', '圖表', '數據分析', '視覺化']
print(type(result))  # <class 'list'>
輸出直接是 Python List，可以直接用於迴圈處理。

實作練習：餐廳評論分析器

from pydantic import BaseModel, Field
from typing import List

class RestaurantReview(BaseModel):
    restaurant_name: str = Field(description="餐廳名稱")
    rating: int = Field(description="評分 1-5")
    pros: List[str] = Field(description="優點")
    cons: List[str] = Field(description="缺點")
    recommend: bool = Field(description="是否推薦")

parser = PydanticOutputParser(pydantic_object=RestaurantReview)

template = PromptTemplate(
    template="""分析以下餐廳評論：
{review}

{format_instructions}""",
    input_variables=["review"],
    partial_variables={"format_instructions": parser.get_format_instructions()}
)

chain = template | llm | parser
result = chain.invoke({
    "review": "鼎泰豐的小籠包皮薄餡多，服務態度很好，但價格偏高要排隊很久"
})
